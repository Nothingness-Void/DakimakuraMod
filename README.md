# Dakimakura Mod (Nothingness-Void Fork)

A maintained fork of Dakimakura Mod for Minecraft Forge. The mod allows players to place Dakimakuras on beds and around the world, and keeps support for custom Dakimakura image packs.

This fork currently focuses on:

- Minecraft Forge 1.20.1 maintenance fixes
- Dedicated server compatibility fixes
- A Forge 1.21.1 port on the `codex/port-1.21` branch
- GitHub Actions builds for both supported versions

## Downloads

Releases are published from this repository:

https://github.com/Nothingness-Void/DakimakuraMod/releases

The `v1.0.0-fix` release includes:

- `dakimakuramod-1.0.0+mc1.20.1.jar`
- `dakimakuramod-1.0.0+mc1.21.1.jar`

## Branches

- `master`: Minecraft Forge 1.20.1
- `codex/port-1.21`: Minecraft Forge 1.21.1

## Building

Use the Gradle wrapper:

```powershell
./gradlew.bat build
```

Build outputs are written to `build/libs/`.
